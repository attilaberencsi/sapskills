# Copilot Instructions

## Local skill locations

A task may benefit from a reusable skill, first check whether the following local skill folders are available in the current session context:

- `D:\Workspaces\ADT_ECLIPSE\.agents\skills\`

If these skill files are accessible in the current environment, read their `SKILL.md` files before making code changes.
List available skills with their path only upon first response in the chat session, confirming you have access to the files. When possible output them as clickable links to be able to open the skill file.
